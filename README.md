# FilmOneri

>  Makine Öğrenmesi dersi dönem projesi.

![Baz Görünüş](resim/durum.png)

**Kullanılan Teknolojiler;**

- NodeJS
- Webpack
- Electron
- TypeScript
- jQuery

Gerekli paketleri kurmak için gerekli komut;
> npm install

Derleyip çalıştırmak için gerekli komut;
> npm run build && npm run start
